  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-3">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Mslicer in minutes
  </p>

  <Steps>
    <Step title="Install Mslicer">
      Download the latest release for your platform from [GitHub Releases](https://github.com/connorslade/mslicer/releases), [Flathub](https://flathub.org/apps/com.connorcode.mslicer), or [Nixpkgs](https://search.nixos.org/packages?channel=unstable&show=mslicer).
      
      ```bash
      # Or build from source
      git clone https://github.com/connorslade/mslicer
      cd mslicer
      cargo build -r -p mslicer
      ```
    </Step>
    
    <Step title="Configure your printer">
      Launch Mslicer and open the **Slice Config** panel. Set your printer's platform resolution and build volume. Defaults are for the ELEGOO Saturn 3 Ultra (11520×5120 px, 218.88×122.88×260 mm).
      
      <Note>
        Incorrect platform settings may cause your printer to reject the sliced file without displaying an error.
      </Note>
    </Step>
    
    <Step title="Import and slice a model">
      Import an STL or OBJ file via **File → Import Model** or drag and drop into the viewport. Position your model, then click **Slice** to generate layers. Export to your printer's format (.goo, .ctb, .nanodlp).
      
      <Tip>
        Try the built-in test model by pressing `Ctrl+T` (Utah Teapot)
      </Tip>
    </Step>
  </Steps>
</div>
